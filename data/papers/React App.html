Answer

A Database Management System (DBMS) is a computerized record-keeping system and a repository or container for a collection of computerized data files. Its overall purpose is to allow users to define, store, retrieve, and update information contained in the database on demand. A database system, which a DBMS facilitates, is a collection of interrelated data and a set of programs that enable users to access and modify this data, providing an abstract view of the data by hiding storage and maintenance details.
